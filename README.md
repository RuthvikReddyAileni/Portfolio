# Ruthvik Reddy — Portfolio

A clean, production-grade personal portfolio website built with HTML, CSS, and vanilla JavaScript.

## 🚀 Quick Start

Just open `index.html` in any modern browser — no build tools or dependencies required.

## 📁 Structure

```
portfolio/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # All styles with CSS variables
├── js/
│   └── main.js         # Interactions, animations, scroll effects
└── README.md
```

## ✨ Features

- **Custom cursor** with smooth trailing animation
- **Scroll-triggered reveal animations** with staggered timing
- **3D tilt effect** on project cards
- **Typewriter animation** in the hero section
- **Scroll progress bar** at the top
- **Active nav highlighting** based on current section
- **Fully responsive** — mobile, tablet, desktop
- **Noise texture overlay** for depth
- **Animated background glows** in hero section

## 🎨 Design

- **Color scheme**: Dark with purple accent (`#7c6af5`)
- **Fonts**: Syne (display) + DM Mono (code) + Fraunces (italic accents)
- **Style**: Dark, editorial, developer-focused

## 🛠 Customization

All color tokens are in `:root` in `style.css` — change `--accent` to rebrand instantly.

To update content, edit the HTML sections:
- `#hero` — Name, tagline, stats
- `#about` — Bio text and card info
- `#experience` — Work timeline
- `#projects` — Project cards
- `#skills` — Skill chips
- `#contact` — Contact links

## 📬 Contact Info Updated
